# Telshe Yeshiva Redesign - Content Questionnaire

To fully build out and polish the new website, I need some specific information from you or the Yeshiva's administration. Please answer the following questions as thoroughly as possible.

## 1. Branding & Messaging
*   **Tagline:** Is there an official tagline or slogan used by the Yeshiva? (Currently using placeholders like *"Carrying the Torch of Torah"*).
*   **Mission Statement:** Do you have an official, finalized mission statement for the About page?
*   **Colors:** The current site uses Navy (`#0f1c2e`) and Gold (`#d4af37`). Are these the exact brand colors, or do you have official hex codes?

## 2. Academics & Divisions
We currently list three main divisions: Mechina, Bais Medrash, and Kollel. 
*   **Kollel Details:** We need the names of the Roshei Kollel, their specific titles, and a brief description of the Kollel’s focus.
*   **Yavne Teachers Seminary:** Does this branch (the women's teacher training program) need its own dedicated page on this website, or does it have a separate site?
*   **Sedarim:** Would you like to publish the daily schedule (Sedarim times, Shacharis, Mincha, Maariv) for the Bais Medrash or Mechina on the site?

## 3. Rebbeim & Faculty Updates
*   **Bios:** Please provide an authentic, 2-3 sentence biography for each Rebbi currently listed.
*   **Images:** We need professional headshots (or preferred photos) for each Rebbi. If photos are not permitted or preferred, should we continue using the circular initials (e.g., "SK" for Shloimy Kohn)?
*   **Additional Faculty:** Are there any Mashgichim, Shoelim U'Meishivim, or administrative faculty (e.g., Executive Director, Menahel) that need to be added to the directory?

## 4. Media & Assets
*   **Campus Photography:** Do you have a shared Google Drive or Dropbox link with high-resolution, recent photos of the Wickliffe campus, Bais Medrash, and dormitories? 
*   **Historical Photos:** Do you have access to archival photos (e.g., European Telshe, early years in Cleveland) to enrich the History section on the About page?

## 5. Alumni & Legacy
*   **Testimonials/Quotes:** Do you have any powerful quotes from notable alumni or Gedolim about the impact of Telshe Cleveland?
*   **Alumni Network:** Should we add an "Alumni" page for network registrations, upcoming Yarchei Kallah events, or reunions?

## 6. Donations & Memorials
*   **Cardknox Integrations:** We currently iframe the main Cardknox donation portal. Are there specific campaigns (e.g., "Building Fund", "Scholarship Fund") that need their own dedicated buttons or landing pages?

## 7. Contact Information
*   **Extensions:** Please provide a list of relevant phone extensions (e.g., General Office, Registrar, Dean of Students).
